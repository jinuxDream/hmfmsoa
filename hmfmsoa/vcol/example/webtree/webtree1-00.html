webTree插件的使用说明

名称：webTree

定义与说明

webtree是基于ul,li标签实现的，实现了树结构视图的功能。

一、要使用，您需要完成以下工作：

1、编写ul,li标签的html代码(适用于静态加载)

2、在页面加载完成时，调用JS，完成对tree的效果渲染，及参数设置

二、可以实现的表格效果（见左边菜单）

三、可选的属性

此控件可以设置的属性有

属性 类型 值 描述 设置方式

target

 字符串
  • _blank(缺省)
设置节点上的超级链接的target

$('#tree").webtree({target:'mainfram'});
enableIcon boolean false(缺省) 是否在节点左边显示图标 $('#tree").webtree({enableIcon:true});

enable_hitbtn

 boolean true(缺省) 是否启用折叠按钮

$('#tree").webtree({enable_hitbtn:true});
enable_selbtn boolean
  • false(缺省)

    •  

是否选择框

同上
expandOne boolean false(缺省)

是否设置：同级节点仅仅允许一个节点处于展开

 同上
selbtn 对象
  • (缺省) {type:'checkbox',name:null,link:null,isthree:true}
 type可以设置'checkbox'，'radio'

通过JS设置，如：$('#tree").webtree({selbtn:{type:'checkbox',name:'menu_id'}});
icons 函数对象 (缺省) {type:0,leaf:'leaficon2',open:'branch-o',closed:'branch-c',level:['icon_def','icon_1','icon_2','icon_3','icon_4']}

type:0，显示分支与叶子两种类型的图标

type:1，根据level的不同层级显示图标

type:2，根据li标签的节点icon_type属性显示图标

按照层级显示图标：$('#tree").webtree({icons:{type:1}});

按照节点类型显示图标：$('#tree").webtree({icons:{type:2}});
itemClass 数组 ['item_def','item_1','item_2','item_3','item_4'] 节点标题的样式

同上

collapsed boolean true(缺省) 树的初始状态是否为折叠状态 $('#tree").webtree({collapsed:false});
loadOnExpand boolean false(缺省) 当节点被展开时，是否动态加载子节点  
loadUrl boolean null(缺省) 加载使用的URL  

 

 

三、可选的方法

函数名称 参数列表 适用情况 说明 调用方法

$.fn.getSelectedNode

 无   取得当前选择的节点  

$.fn.expand(node,toExp)

node要展开的节点，

toExp，true展开，false折叠

 展开指定的节点 如果node为空，则展开整个树  
$.fn.addNode(newNode,posNode,ischild)

newNode，新节点的html串posNode，插入位置

ischild：true：作为posNode的子节点；false：作为posNode的兄弟节点

 添加节点    
$.fn.removeNode(node)   删除节点    
         
         
$.fn.gridRowspan(column)        
$.fn.gridColspan(rownum)        
$.fn.loadNode(op,node) node：可选 要加载数据归属的节点，如果忽略则为树的根节点    

op

 

url:必须提供

 请求的URL    

resultName:'result'，可选

 返回json数据的名字  
success:null，可选

服务器成功返回后，节点加载前的回调函数，

回调函数的参数为：function allback(data){}，data为服务器返回的json数据

  
mapName:

用于存放服务器端返回json数据项名字的对应关系，mapName:{id:"id",name:"name",isleaf:"isleaf",

url:"url",parent_id:"parent_id",param:"param"}

  
isfull:false可选 是否加载整棵树  
其他有效参数， 参见$.webutil.submitByAjax的op参数。

特别注意：服务器返回的JSON数据中的parent_id不能引用在当前行之前未出现的节点ID，如果第i行的parent_id对应的节点在第i行之后，则无法显示！

id

 节点的唯一ID  
name 节点的文本内容  
url 节点被点击时执行的请求URL（可选，可以不填）  
level level（可选，可以不填）  
isleaf 是否是叶子节点（可选，可以不填）  
parent_id 归属的父节点ID
