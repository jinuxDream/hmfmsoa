按钮控件的使用说明

名称：PicButton

定义与说明

PicButton图片按钮插件是基于标准的<button>标签的功能扩展，支持<button>标签原有的所有属性、事件，实现自定义的图片效果的按钮。

要使用PicButton，需要完成以下步骤：

1、在文档初始化完毕后，编写图片按钮的初始化JS代码

<script type="text/javascript"> 
$(document).ready(function(){	
	$('.btn').PicButton();
});		
</script>

2、需要显示按钮的地方，写按钮的html代码

<button class="btn" id ='ddd' type="button" value="1" onclick="form1.submit();"  mask > 新增</button>

注释：

1.PicButton可以放置在页面的任何地方使用
2.PicButton可以放置在 <div class="btn_toolbar" >内，与BtnToolBar插件结合起来使用
3.PicButton可以与BtnToolBar插件结合起来使用，实现工具条的功能效果

提示：请将PicButton的初始化语句放置在页面加载完毕后调用

详细描述

PicButton插件可以完成以下功能：

一、用按钮执行Javascript， 详细参见示例代码

二、用按钮执行表单提交，并页面跳转 详细参见示例代码

三、用按钮执行页面跳转，不提交表单 详细参见示例代码

四、与BtnToolBar组合使用，完成按钮工具条的自动隐藏与显示效果。 详细参见示例代码

可选的属性

此控件可以设置的属性有

属性 值 描述 设置方式
btn_href
  • string(缺省为空)
 按钮被点击时，需要提交的URL，设置此属性后，按钮将会完成以下事情： 1.进行表单验证 2.设置关联表单的action 3.调用表单的submit() 通过元素的属性设置，如：
<button class="btn" id ='ddd' type="button" btn_href="someaction.do" istip="1" nosubmit="1"> 新增</button>
istip
  • false(缺省)
  • true
 按钮被点击时，是否提示确认框（确认框的内容是”您确认要<按钮的名字>吗？"），当btn_href未设置或issubmit="0"时，此属性无效

通过JS或元素的属性设置，如：
$('.btn').PicButton({istip:true}); 或设置属性 istip="1" （提示）， istip="0" （不提示）
issubmit
  • true(缺省)
  • false
 在页面跳转时，是否需要提交表单，true提交表单，false不提交表单。当btn_href未设置时，此属性无效,当通过属性设置时，issubmit="1"时，此值为true,issubmit="0"时，此值为false 通过js方式或元素的属性设置，如：
var by=$('body');
$('.btn').PicButton({issubmit:true});或设置属性 issubmit="1" （提交）， issubmit="0" （不提交）
ischeck
  • true(缺省)
  • false
 当提交表单时，true检查验证表单，false不检查验证表单。当btn_href未设置时，此属性无效,当通过属性设置时，ischeck="1"时，此值为true,ischeck="0"时，此值为false 通过js方式或元素的属性设置，如：
var by=$('body');
$('.btn').PicButton({ischeck:true});或设置属性 ischeck="1" （验证）， ischeck="0" （不验证）
popup
  • false(缺省)
  • true
 页面跳转时，以弹出窗口方式显示,当btn_href未设置时，此属性无效

通过js方式或元素的属性设置

或设置属性 popup="1" （弹出）， popup="0" （不弹出）
formobj
  • null(缺省)
  • jquery对象
 按钮点击时，要提交的表单对象（jquery对象） ，当btn_href未设置时，此属性无效 通过js方式设置，如：
var frm1=$('#form1');
$('.btn').PicButton({formobj:frm1});
maskobj
  • null(缺省)
  • jquery对象
 按钮点击时，需要遮罩的对象（jquery对象），默认是body对象，当btn_href未设置时，此属性无效 通过js方式设置，如：
var by=$('body');
$('.btn').PicButton({ formobj:by});
mask
  • false(缺省)
  • true(当设置属性mask="1"时 此属性为true)
 此属性被废弃 通过js方式或元素的属性设置，如：
<button class="btn" id ='ddd' type="button" mask="1" > 新增</button>
force
  • false(缺省)
  • true(当设置属性force="1"时 此属性为true)
 与BtnToolBar插件结合使用时，是否强制显示此按钮(不收按钮显示条件的限制），按钮不再BtnToolBar显示时，此属性无效 通过js方式或元素的属性设置，如：
<button class="btn" id ='ddd' type="button" force="1" > 新增</button>
enabIcon true(缺省)，可选 是否启用按钮图标  
icon null,空 用于设置某个按钮的图标式名称，如果未设置，则采用“btn_icon_def”样式， 点此查看webui78支持的按钮图标 。推荐尽量使用全局名称对应关系设置，参见：webUtil.setting.btn_icons

实例

图片按钮的一般用法：初始化

<script type="text/javascript"> 
$(document).ready(function(){	
	$('.btn').PicButton();

});		
</script>

需要显示按钮的地方，需要如下写法

<button class="btn" id ='ddd' type="button" value="1" onclick="form1.submit();"  mask > 新增</button>

注释： 上述代码中的class="btn" 中的class名字可以是任意自定义的类名称
